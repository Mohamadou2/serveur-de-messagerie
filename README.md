# 📧 Configuration d'un Serveur de Messagerie

Projet d'Administration Système — DA2I.

## 📋 Description

Déploiement et configuration d'un serveur de messagerie complet
sous **CentOS Linux** en environnement virtualisé (VirtualBox).

> ⚠️ Les fichiers de configuration originaux ont été perdus suite à
> une panne matérielle. Ce README documente l'architecture et les
> étapes réalisées.

## 🏗️ Environnement

- **OS :** CentOS (Linux)
- **Virtualisation :** Machine virtuelle
- **MTA :** Postfix (SMTP) + Sendmail

## ⚙️ Étapes réalisées

1. Installation et configuration de **Postfix** comme MTA principal
2. Configuration de **Sendmail** pour la compatibilité
3. Paramétrage des enregistrements **DNS** (MX, A, PTR)
4. Tests d'envoi et réception de mails locaux et distants
5. Configuration des règles de sécurité de base

## 🛠️ Technologies

- CentOS Linux · Postfix · Sendmail · DNS · VirtualBox
